# 🏛️ Philosopher Survivor

철학자들의 생존 게임 — Vampire Survivor 스타일의 철학 테마 웹 게임

## 🎮 캐릭터
- **플라톤** (이데아주의): 근접 광역 공격 특화, 화염 속성
- **아리스토텔레스** (경험주의): 관통/원거리 공격 특화, 냉기 속성

## ⚔️ 스킬 등급
| 등급 | 확률 | 색상 |
|------|------|------|
| 보통 | 55% | 회색 |
| 레어 | 30% | 파랑 |
| 유니크 | 10% | 보라 |
| 에픽 | 5% | 금색 |

## 🕹️ 조작법
- **이동**: WASD 또는 방향키
- **스킬 선택**: 마우스 클릭 또는 방향키 + Enter

## 🚀 실행 방법
`index.html` 파일을 브라우저로 열면 바로 플레이 가능합니다.
