# 🏛️ 철학의 미궁: 캐릭터 스프라이트 시트 제작 가이드 및 요구사항

이 문서는 게임 내 캐릭터 스프라이트가 깨지거나 잘못 렌더링되는 현상을 해결하기 위해 필요한 **스프라이트 시트의 정확한 규격과 양식**을 기술합니다.

제공해주시는 새로운 스프라이트 시트가 아래의 규격에 정확히 맞추어 제작되면, 게임 코드 수정 없이 파일 교체만으로 완벽하고 부드러운 애니메이션이 적용됩니다.

---

## 1. 📐 전체 이미지 규격 및 해상도

게임은 **가로 12칸, 세로 8칸(12x8 Grid)**으로 나누어진 정밀한 그리드 시스템을 사용합니다. 캐릭터의 크기에 따라 두 가지 해상도 중 하나를 선택하여 제작해 주십시오.

> **💡 권장 규격 (128x128px 셀):** 해상도가 선명하고 정렬하기 가장 편리한 규격입니다.

| 구분 | 1개 셀 크기 (정사각형) | 전체 이미지 권장 해상도 | 비고 |
| :--- | :--- | :--- | :--- |
| **권장 규격** | **`128 x 128 px`** | **`1536 x 1024 px`** | 고해상도 pixel-art 및 그래픽 정렬에 가장 적합 |
| **초고해상도** | **`256 x 256 px`** | **`3072 x 2048 px`** | 보다 정밀하고 큰 사이즈의 그래픽용 |

### ⚠️ 필수 주의 사항
1. **정중앙 정렬 (Center Alignment):** 캐릭터 본체는 각각의 가로세로 정사각형 셀의 **정중앙**에 배치되어야 합니다.
2. **범위 초과 금지 (No Overflow):** 캐릭터의 머리 끝, 발 끝, 무기 등의 그래픽 요소가 해당 셀의 경계선(예: 128px 또는 256px 경계)을 단 1픽셀이라도 벗어나 이웃 셀을 침범해서는 안 됩니다. (침범 시 잘리거나 다른 캐릭터 모션에 섞여 나오게 됩니다.)

---

## 2. 🗺️ 캐릭터 슬롯 배치도 (12x8 그리드 구조)

하나의 스프라이트 시트에는 **최대 8개의 캐릭터**를 배치할 수 있는 구조(가로 4명 x 세로 2명)입니다. 각 캐릭터는 **3열(모션) x 4행(방향)**의 영역을 차지합니다.

현재 게임 코드(`game.js`)에 설정된 **학파 진화 단계별 캐릭터 배치 슬롯 위치**는 다음과 같으므로, 이 구조에 맞추어 이미지를 배치해 주셔야 합니다.

```text
┌──────────────────────────────────────── 가로 전체 12 컬럼 (0 ~ 11) ────────────────────────────────────────┐
│                                    │                                    │                                    │
│       [Slot 0]                     │       [Slot 1]                     │       [Slot 2]                     │       [Slot 3]                     │
│       비워둠 (사용 안함)             │       1단계 캐릭터 (시작)             │       2단계 캐릭터                 │       3단계 캐릭터                 │
│       - 가로: 0, 1, 2열             │       - 가로: 3, 4, 5열             │       - 가로: 6, 7, 8열             │       - 가로: 9, 10, 11열           │
│       - 세로: 0, 1, 2, 3행          │       - 세로: 0, 1, 2, 3행          │       - 세로: 0, 1, 2, 3행          │       - 세로: 0, 1, 2, 3행          │
│                                    │                                    │                                    │
├────────────────────────────────────┼────────────────────────────────────┼────────────────────────────────────┼────────────────────────────────────┤ 세로 0 ~ 3행 (상단)
│                                    │                                    │                                    │
│       [Slot 4]                     │       [Slot 5]                     │       [Slot 6]                     │       [Slot 7]                     │
│       비워둠 (사용 안함)             │       4단계 캐릭터                 │       5단계 캐릭터                 │       6단계 캐릭터 (최종)           │
│       - 가로: 0, 1, 2열             │       - 가로: 3, 4, 5열             │       - 가로: 6, 7, 8열             │       - 가로: 9, 10, 11열           │
│       - 세로: 4, 5, 6, 7행          │       - 세로: 4, 5, 6, 7행          │       - 세로: 4, 5, 6, 7행          │       - 세로: 4, 5, 6, 7행          │
│                                    │                                    │                                    │
└────────────────────────────────────┴────────────────────────────────────┴────────────────────────────────────┴────────────────────────────────────┘ 세로 4 ~ 7행 (하단)
```

### 👤 학파별 진화 캐릭터 위치 매핑 매뉴얼

#### 1) 🔥 이상주의 (Idealism) 시트 배치 구조 (`plato_sprite_sheet_clean.png`)
* **Slot 0**: 비워둠 (혹은 임의 이미지)
* **Slot 1 (플라톤)**: 가로 3, 4, 5열 / 세로 0, 1, 2, 3행
* **Slot 2 (스토아 학파)**: 가로 6, 7, 8열 / 세로 0, 1, 2, 3행
* **Slot 3 (아우구스티누스)**: 가로 9, 10, 11열 / 세로 0, 1, 2, 3행
* **Slot 4**: 비워둠
* **Slot 5 (데카르트)**: 가로 3, 4, 5열 / 세로 4, 5, 6, 7행
* **Slot 6 (칸트)**: 가로 6, 7, 8열 / 세로 4, 5, 6, 7행
* **Slot 7 (사르트르)**: 가로 9, 10, 11열 / 세로 4, 5, 6, 7행

#### 2) ❄️ 경험주의 (Empiricism) 시트 배치 구조 (`aristotle_clean.png`)
* **Slot 0**: 비워둠 (혹은 임의 이미지)
* **Slot 1 (아리스토텔레스)**: 가로 3, 4, 5열 / 세로 0, 1, 2, 3행
* **Slot 2 (에피쿠로스)**: 가로 6, 7, 8열 / 세로 0, 1, 2, 3행
* **Slot 3 (토마스 아퀴나스)**: 가로 9, 10, 11열 / 세로 0, 1, 2, 3행
* **Slot 4**: 비워둠
* **Slot 5 (베이컨)**: 가로 3, 4, 5열 / 세로 4, 5, 6, 7행
* **Slot 6 (벤담 / 밀)**: 가로 6, 7, 8열 / 세로 4, 5, 6, 7행
* **Slot 7 (듀이)**: 가로 9, 10, 11열 / 세로 4, 5, 6, 7행

## 3. 🏃 개별 슬롯 내부 모션 및 방향 규칙 (3x4 구조)

각각의 캐릭터가 위치하는 개별 슬롯(3열 x 4행) 내부는 아래 규칙에 따라 애니메이션이 구성됩니다.

### 🔄 세로 행 (바라보는 방향)
* **0번째 행**: **정면(아래)**을 향해 서거나 걷는 모션
* **1번째 행**: **왼쪽**을 향해 서거나 걷는 모션
* **2번째 행**: **오른쪽**을 향해 서거나 걷는 모션
* **3번째 행**: **뒷면(위)**을 향해 서거나 걷는 모션

### 👣 가로 열 (걷기 애니메이션 프레임)
* **0번째 열**: 왼발을 앞으로 내딛는 걷기 모션
* **1번째 열**: 차려 자세 (가만히 서 있는 기본 대기/Idle 모션)
* **2번째 열**: 오른발을 앞으로 내딛는 걷기 모션

> **💡 움직임 흐름:** 플레이어가 이동할 때 게임 내에서는 `0열(왼발) ➡️ 1열(정지) ➡️ 2열(오른발) ➡️ 1열(정지)` 프레임 순으로 순환하며 부드러운 걷기 모션이 완성됩니다.
