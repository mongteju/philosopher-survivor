# 📊 이세계 철학자 키우기 - 게임 밸런스 데이터 시트

이 문서는 게임 내 모든 스킬, 패시브, 일반 몬스터, 보스 몬스터의 능력치와 데이터를 정리한 밸런스 시트입니다. 
여기에 기입된 값을 수정하고 저에게 **"시트 수정했으니 반영해줘"**라고 하시면, 제가 이 마크다운 표의 데이터를 읽어서 실제 게임 코드(`game.js`)에 자동으로 반영해 드립니다!

## 4. 보스 몬스터 데이터 (Boss Monster Stats)

* 보스 체력 공식: `기본 체력 * (1 + (플레이어 레벨 - 1) * 0.10)`
* 보스 공격 시 플레이어가 받는 대미지: 일반 충돌 시 `22` (보스 탄막 적중 시 `18`, 워닝 존 적중 시 `30~35`)
* 보스는 체력이 50% 미만으로 떨어지면 **'광분 상태(Phase 2)'**에 진입하여 **이동 속도가 1.5배 빨라지고, 탄막 쿨타임이 약 35% 단축**됩니다.

| :---: | :--- | :--- | :---: | :---: | :---: | :--- | :--- |
| **Stage 1** | 소피스트 | 고대 그리스 | 3,000 | 1.2 | 38 | 나선형 퍼지는 탄막 (8방향) | 탄막 개수 8 -> 12개로 증가 |
